## ベクトルへ変換するステップ

 1. テキストのクリーニング : テキスト中のノイズを除去する(htmlタグの除去)
 2. 単語の分割(形態素解析)
 3. 単語の正規化
 4. ストップワードの除去
 5. 単語のベクトル表現
 6. 文書のベクトル表現
